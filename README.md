# Свадебное приглашение Динара и Виктории

Это простой статический сайт для GitHub Pages. Он работает без React, Vite, npm,
серверов и сборки. Его можно открыть двойным кликом по `index.html`.

## Файлы

- `index.html` - структура сайта и тексты.
- `styles.css` - внешний вид, цвета, адаптация под телефон и компьютер.
- `script.js` - лёгкие анимации и аккуратная обработка отсутствующих фотографий.
- `assets/images` - папка для фотографий и схем проезда.

## Как заменить фото

Положите изображения в папку `assets/images` с такими именами:

- `hero.jpg` - большое фото для первого экрана.
- `photo-1.jpg`
- `photo-2.jpg`
- `photo-3.jpg`

Если какого-то изображения нет, сайт не ломается: вместо него появится аккуратная
карточка-заглушка.

Лучше использовать JPG-файлы шириной примерно 1200-1800 пикселей. Так сайт будет
красиво выглядеть и не станет слишком тяжёлым для телефона. Сейчас сайт
использует `hero.jpg` как фон первого экрана, а `photo-1.jpg`, `photo-2.jpg` и
`photo-3.jpg` - в галерее.

## Где лежат ссылки на Яндекс.Навигатор

Откройте `index.html` и найдите блок `Как добраться`.

Ссылки стоят прямо в `href` у кнопок “Открыть в Яндекс.Навигаторе”:

- для Дома торжеств “Сафиса”:
  `yandexnavi://build_route_on_map?lat_to=55.729306&lon_to=37.543740`
- для ресторана “Град Южный”:
  `yandexnavi://build_route_on_map?lat_to=55.809960&lon_to=37.323617`

## Где поменять координаты

Координаты указаны в двух местах в `index.html`:

- в блоке `Локации`;
- в блоке `Как добраться`, рядом с кнопками и в атрибуте
  `data-copy-coordinates`.

Если меняете координаты, замените их и в видимом тексте, и в ссылке
`yandexnavi://...`, и в `data-copy-coordinates`.

## Как заменить контакты

В `index.html` в блоке “Контакты” уже указана ссылка на Telegram:

```html
https://t.me/MrDinar
```

Для MAX и телефона замените placeholders:

- `MAX_LINK`
- `PHONE_LINK`

Для телефона обычно используется формат:

```html
tel:+79991234567
```

## Как открыть сайт локально

Откройте файл `index.html` двойным кликом. Он должен открыться в браузере.

Также можно открыть через PowerShell:

```powershell
Start-Process .\index.html
```

## Как опубликовать на GitHub Pages

1. Создайте репозиторий на GitHub.
2. Загрузите в него файлы `index.html`, `styles.css`, `script.js`, `README.md`
   и папку `assets/images`.
3. В репозитории откройте `Settings`.
4. Перейдите в раздел `Pages`.
5. В блоке `Build and deployment` выберите:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - папка: `/root`
6. Нажмите `Save`.

Через несколько минут GitHub покажет ссылку на опубликованный сайт.
